# 认证说明

## 应用信息

- 应用名称：身份证照片合成助手
- 版本号：v1.0
- 目标平台：Windows 10 (版本 1809 及以上)、Windows 11
- 架构：x64

## 功能概述

本工具是一款桌面端身份证照片处理工具，用户选择身份证正反面照片后，程序自动进行边缘检测、透视校正、方向识别并合成为一张图片，可选添加文字水印和 A4 排版输出。

## 兼容性说明

- 本工具为标准的 Win32 桌面应用程序，使用 PyInstaller 打包为独立 EXE
- 不依赖任何运行时环境（已内置 Python 运行时）
- 不修改系统注册表、不安装系统服务、不写系统目录
- 不在后台运行，无开机自启动
- 不需要管理员权限即可运行
- 不需要网络连接

## 使用的系统 API

| API | 用途 |
|-----|------|
| `SetProcessDpiAwareness` | 高 DPI 显示适配 |
| `GetModuleHandleW` | 窗口图标加载 |
| drag-and-drop (windnd) | 支持文件拖拽导入 |

以上 API 均为标准 Windows 桌面应用常用调用，不涉及任何敏感系统功能。

## 隐私与安全

- 完全离线运行，无任何网络通信
- 不收集、不上传任何用户数据
- 使用 OpenCV Haar Cascade 本地人脸检测算法，仅用于方向判定，人脸数据不保存、不传输
- 不包含任何第三方 SDK、广告或分析服务

## 开源许可

本工具使用的第三方开源库均采用 MIT、BSD 或 Apache 2.0 等宽松许可协议，详情见软件内"帮助 > 关于"页面。
